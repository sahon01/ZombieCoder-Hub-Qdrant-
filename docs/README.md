# Documentation Structure

This directory contains all the documentation for the UAS Admin Panel project, organized by category.

## Directory Structure

- **api/** - API documentation and specifications
- **architecture/** - System architecture and design documents
- **configuration/** - Configuration guides and settings
- **development/** - Development guides, coding standards, and Next.js related documentation
- **deployment/** - Deployment guides, including Vercel documentation
- **installation/** - Installation guides and quick start documentation
- **testing/** - Testing documentation and procedures
- **admin_comments/** - Administrative comments, implementation summaries, and internal documentation
